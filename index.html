Visual Studio

Visual Studio is an app to code on using, html, css and javascript. Its a really good app if you want to learn coding and its pretty easy to use.

How to set up?

Setting up isn't that hard, just download the visual studio app using this link - https://code.visualstudio.com/Download - then make a folder in whichever laptop or desktop your using. When you open the visual studio app you will see the option "open", click on it and open ur folder. Add the html, css and javascript elements and your ready to code!

Console Screen

The console screen is where we see our code and what we've done, we'll also see any errors we've done there incase our code isn't working.

How to open the console screen?

On ur output screen, double click using either mouse or keypad, then there will be an option to "inspect", click that. After that you will see a option of "console", if u click on that you will reach the console screen.
